# yuen-create-app

## Install

[npm package](https://www.npmjs.com/package/yuen-create-app)

```bash
// npm
npm install -g yuen-create-app

// yarn
yarn global add yuen-create-app
```

## Create a project
```bash
yuen create my-project
```
